# 나의 업무 관리 (PMS)

회사 업무 · 오늘 꼭 해야 하는 일 · 개인 업무 · 자기 계발 · 개인 일정을
**5개 영역으로 나눠서** 관리하는 가벼운 개인 대시보드입니다.

## 실행 방법 (권장)

**`start-pms.bat` 더블클릭** → 검은 창이 뜨고 잠시 후 브라우저가 자동으로 열립니다.
또는 바탕화면의 **「나의 업무 관리」 바로가기**를 더블클릭하세요.

- 사용하는 동안 검은 창은 **그대로 두세요.** 다 쓰면 창을 닫으면 종료됩니다.
- 이렇게 실행하면 데이터가 이 폴더의 **`pms-data.json` 실제 파일**에 저장됩니다.
  (브라우저 캐시를 지워도 안전하고, OneDrive 폴더라 자동으로 백업·동기화됩니다.)

> 참고: `index.html`을 그냥 더블클릭해도 열리지만, 그땐 데이터가 브라우저(localStorage)에만
> 저장됩니다. 안정적으로 쓰려면 `start-pms.bat`(바로가기) 사용을 권장합니다.

## 기능

- **5개 영역 패널**: 🏢 회사 업무 · 🔥 오늘 꼭 해야 하는 일 · 🏠 개인 업무 · 📚 자기 계발 · 📅 개인 일정
- **빠른 추가**: 각 패널 아래 입력칸에 쓰고 Enter
- **완료 체크**: 왼쪽 체크박스 클릭 → 완료(취소선) 후 아래로 정렬
- **상세 편집**: 항목 클릭 → 내용·메모·분류·우선순위·날짜·완료여부 수정 (분류 변경 시 영역 이동)
- **드래그 이동**: 항목을 끌어다 다른 영역으로
- **날짜 강조**: 오늘 마감=주황, 마감 지남=빨강
- **요약 바**: 오늘 할 일 / 오늘 마감 / 전체 완료 현황
- **완료 숨기기** 토글, **백업 내보내기/복원**(JSON)

## 데이터 백업

평소 저장은 자동입니다(`pms-data.json`). 더 안전하게 보관하려면 화면 우측 상단
**`⬇ 백업`** 으로 JSON 파일을 따로 내려받아 두세요. (복원은 **`⬆ 복원`**)

## 구성 파일

| 파일 | 역할 |
|------|------|
| `index.html` | 화면 구조 |
| `style.css`  | 디자인 |
| `app.js`     | 동작 로직 |
| `server.py`  | 로컬 서버 (파일로 저장) |
| `start-pms.bat` | 실행 파일 (서버 시작 + 브라우저 열기) |
| `pms-data.json` | 내 데이터 (자동 생성) |
