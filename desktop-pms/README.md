# 🗂️ 비엘테크(주) 경영 대시보드

대표이사용 개인 PMS · 데이터는 **내 컴퓨터의 파일(`data.json`)에 저장**됩니다.

## ▶ 실행 방법 (이것만 기억하세요)

**`대시보드 실행.bat`** 파일을 **더블클릭**하세요.
- 검은 창이 하나 뜨고, 잠시 후 브라우저에 대시보드가 자동으로 열립니다.
- **이 검은 창은 닫지 마세요.** (프로그램 본체입니다. 창을 닫으면 종료됩니다.)
- 다 쓰면 브라우저 탭을 닫고, 검은 창도 닫으면 됩니다.

> 바탕화면에 바로가기를 만들고 싶으면: `대시보드 실행.bat` 우클릭 → "바로 가기 만들기" → 만들어진 바로가기를 바탕화면으로 옮기세요.

## 💾 데이터 저장 방식

- 입력/수정하는 모든 내용은 자동으로 이 폴더의 **`data.json`** 파일에 저장됩니다.
- 좌측 하단에 **`💾 파일에 저장됨`** 이라고 표시되면 정상입니다.
- 이 폴더는 OneDrive 안에 있어 **클라우드에도 자동 백업**됩니다.
- `data.bak.json` 은 직전 상태 자동 백업본입니다(실수 복구용).
- 브라우저 캐시를 지워도 데이터는 `data.json` 에 남아 있어 사라지지 않습니다.

### ⚠️ 주의
`index.html` 을 그냥 더블클릭해서 열면(파일 경로가 `file://...`) 파일 저장이 안 되고
**브라우저에만 임시 저장**됩니다. (좌측 하단에 `⚠️ 브라우저에만 저장` 표시)
**반드시 `대시보드 실행.bat` 으로 여세요.**

## 메뉴 구성

| 메뉴 | 내용 |
|------|------|
| 📊 대시보드 | 핵심 지표 + 오늘 할 일·일정·출고/입금·자금 요약 |
| ✅ 오늘 할 일 | 우선순위별 체크리스트 |
| 🗂️ 개인 업무 | 상태(할 일/진행/완료)·우선순위·마감일 |
| 📅 개인 일정 | 미팅·출장·행사 일정 |
| 💰 회계·자금 현황 | 수입/지출 장부 + 잔액 자동 누계 |
| 🌐 해외주문/출고·입금 | 오더별 출고·입금 상태, 잔액 자동계산 |
| 👥 1인당 매출 공헌도 | 인력별 매출 기여 차트 |
| 🏭 제조비용 | 비목별 구성 + 제조원가율 |

## 백업 / 복원 / 초기화

- 화면 맨 아래 **전체 백업(JSON)** : 현재 데이터를 파일로 내려받기
- **복원** : 백업 파일에서 되돌리기
- **초기화** : 샘플 데이터를 지우고 빈 상태로 시작 (처음 실제 사용 시 한 번 누르세요)

## 파일 설명

| 파일 | 용도 |
|------|------|
| `대시보드 실행.bat` | ▶ 실행 런처 (이것을 더블클릭) |
| `index.html` | 대시보드 화면 본체 |
| `server.py` | 로컬 서버 (데이터 파일 저장 담당) |
| `data.json` | **내 데이터** (실행 후 자동 생성) |
| `data.bak.json` | 자동 백업본 |
| `personal-board.html` | 처음 만든 개인용 칸반 보드(참고용) |

> 실행에는 Python 이 필요합니다. (이미 설치되어 있습니다.)
