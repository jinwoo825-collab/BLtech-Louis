# 나의 프로젝트 보드 (개인 프로젝트 관리 시스템)

설치·빌드 없이 **브라우저에서 바로 쓰는** 가벼운 칸반 보드입니다.
일할 때 옆에 띄워두고 할 일을 관리하세요.

## 실행 방법

`index.html` 파일을 **더블클릭**하면 기본 브라우저에서 열립니다.
(자주 쓴다면 브라우저 즐겨찾기/북마크에 추가해두면 편해요.)

## 주요 기능

- **프로젝트별 보드** — 왼쪽 사이드바에서 프로젝트를 추가/선택/이름변경/삭제
- **3단계 칸반** — `할 일` → `진행 중` → `완료`
- **카드 드래그** — 카드를 끌어다 다른 칸으로 이동
- **상세 편집** — 카드를 클릭하면 제목·메모·우선순위·마감일·상태 편집창이 열림
- **빠른 추가** — 각 칸 아래 입력칸에 제목을 쓰고 Enter
- **우선순위 색상** — 높음(빨강)/보통(파랑)/낮음(회색)
- **마감일 표시** — 마감이 지난 미완료 카드는 빨간색으로 강조
- **진행률 바** — 프로젝트별 완료 비율 표시

## 데이터 저장 / 백업

- 모든 데이터는 **이 브라우저(localStorage)** 에 자동 저장됩니다. 껐다 켜도 유지돼요.
- 단, 브라우저를 바꾸거나 캐시를 지우면 사라질 수 있으니 가끔 **`⬇ 백업 내보내기`** 로
  JSON 파일을 저장해두세요. 복원은 **`⬆ 백업 가져오기`** 로 그 파일을 선택하면 됩니다.

## 구성 파일

| 파일 | 역할 |
|------|------|
| `index.html` | 화면 구조 |
| `style.css`  | 디자인/색상 |
| `app.js`     | 동작 로직 (저장·드래그·편집 등) |

순수 HTML/CSS/JavaScript로만 되어 있어 외부 라이브러리나 인터넷 연결이 필요 없습니다.
